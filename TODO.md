# Task List App

## What it should do

-   Create tasks (✔)
-   Create reminders (✔)
-   Calculate points for a given day (✔)
-   Ability to set a due date (✔)
-   Ability to check as complete (✔)
-   Search bar (by category, name) (✔)

## Extra features

-   Automatically carry over to new day (✔)
-   Add chime when task is complete (✔)
